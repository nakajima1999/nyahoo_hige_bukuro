# Nyahoo!ひげ袋
## 概要
- 家の中で拾った猫様のひげの
**場所，日にち，長さ，特徴**などを入力して，
投稿する**猫様のひげ専用ブログ**

- 開発環境
    - Python
    - Django
    - Bootstrap
    
- 作った目的
    - 猫様との思い出を残す
    - Djangoを用いた開発の経験を積む
    - 就職活動におけるポートフォリオとして
